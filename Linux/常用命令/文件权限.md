18、chmod 命令
授权与权限 
r: read
w: write
x: execute

u: user
g: group
o: other
chmod u+w aaa.txt
chmod g+w aaa.txt
chmod o+w aaa.txt


20、chown 命令
修改文件或目录的所属的用户或组
chown yakir:wheel x1.txt
      用户  组    文件或目录
